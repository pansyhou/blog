## 文本属性

### 对齐文本

`text-align`只能属性文本内容为**水平对齐方式**

1. left（左对齐
2. right（右对齐
3. center（居中对齐



### 装饰文本

`text-decoration`给文本修饰，下划线、删除线、上划线
